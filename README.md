# Personal Website

This repository contains my personal website and professional portfolio.

The site is intended to serve as my long-term home on the web, showcasing my software projects, build logs, career growth, and future creative or business ventures.

## Goals

- Present a clear professional identity
- Showcase practical software and web projects
- Document what I am learning and building
- Provide an easy way to contact me
- Grow over time without becoming cluttered

## Version 1 Scope

Version 1 will focus on a small, clean static website with:

- Home
- About
- Projects
- Build Log
- Resume
- Contact
