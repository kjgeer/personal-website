# Design System

## Goals

The design should be:

- Clean
- Modern
- Minimalist
- Fast
- Professional
- Accessible
- Responsive
- Easy to maintain
- Dark mode first
- Timeless rather than trendy

## Philosophy

The design should emphasize content over decoration.

Every visual element should have a purpose.

The website should feel calm, organized, and easy to explore.

Visitors should remember the work being showcased, not unnecessary visual effects.

## Motion

Animations should be subtle.

Animations should improve usability, not distract from content.

Avoid excessive movement and large animated backgrounds.

## Performance

Performance is a feature.

The website should:

- Load quickly.
- Optimize images.
- Minimize JavaScript.
- Prefer static content whenever possible.

## Accessibility

The website should be usable by everyone.

Goals include:

- Semantic HTML
- Keyboard navigation
- Good color contrast
- Descriptive alt text
- Visible focus states
