# Site Architecture

## Purpose

This document describes how the website is organized and how visitors navigate through it.

---

## Primary Navigation

- Home
- About
- Projects
- Build Log
- Resume
- Contact

---

## Site Hierarchy

Home
├── About
├── Projects
│   ├── Individual Project Pages
│   └── Project Archive
├── Build Log
├── Resume
└── Contact

---

## Design Philosophy

The website should present a single, cohesive story rather than a collection of unrelated projects.

Projects are organized by purpose, not by programming language.

Games, business software, and learning projects all live within the Projects section.
